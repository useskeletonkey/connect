# Skeleton Key

Connect your AI assistant to your Skeleton Key trips and give it Skeleton
Key's scheduling know-how.

This plugin bundles two things:

- An MCP server connection to the Skeleton Key API, so your assistant can
  read and edit your trips (games, schedules, players, costs, and more).
- The Skeleton Key trip-scheduling skill, so your assistant knows how to
  plan an escape-room trip well (clustering games by neighborhood, leaving
  buffer for drive time, sequencing bookings around confirmation deadlines).

## Install

In Claude Code, run:

```
/plugin marketplace add useskeletonkey/skeleton-key-plugin
/plugin install skeleton-key@skeleton-key
```

Sign in when prompted (OAuth) the first time.
